# Neoskye
An opinionated, open-source, very cool game engine.  
We have a [Discord](https://discord.gg/mtrzeMez)!

## Building

### Linux

You'll need these libraries (+ their development headers) to build [SFML](https://www.sfml-dev.org/)

- freetype
- x11
- xrandr
- udev
- opengl
- flac
- ogg
- vorbis
- vorbisenc
- vorbisfile
- openal
- pthread
